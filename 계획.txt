# ImageViewr 웹서비스

필수 : OOP 를 준수 유지보수성 및 확장성 고려

## 사용 언어
BackEnd : c#
FrontEnd : React(휴대폰으로 가능하도록 반응형 UI작성)

## 아키텍쳐
* 모듈러 모놀리식 아키텍쳐
* 마이크로 서비스 아키텍쳐
* RabbitMq 메시징 시스템
* Unit 테스트 엄격하게 준수하면서

## 기능
* 회원 가입 (인증 서비스, 엑세스 토큰 및 리프레시토큰을 이용)
* 회원 단위로 파일을 업로드 ( 회원 단위로 내가 올린 파일 리스트 볼수 있음, 미리보기기능 지원)
* 회원이 서로 이미지 볼수있도록 요청하고 승낙하면 승낙한 회원의 사진 볼 수 있음 
* 미리보기 기능 사용자가 몇개를 볼지 설정 
* 프론트엔드에서 사진 미리보기전 블러처리로 어떤 UI인지 감은 잡히도록
